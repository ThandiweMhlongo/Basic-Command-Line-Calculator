# Basic-Command-Line-Calculator
Basic Command Line Calculator (Java): Performs basic arithmetic operations. This project is designed to demonstrate basic Java syntax, user input handling, and control flow.

## Features
* Addition(+)
* Subtraction(-)
* Multiplication(*)
* Division(/)

## Prerequisites
To run this project, you need to have the Java Development Kit(JDK) installed on your machine.
* JDK 8 or higher is recommended.
* Verify your installation by running java -version in your terminal.

## How to Run
1. Clone the repository (or download the calculator.java file):
git clone:
2. Navigate to the project directory
3. Compile the Java file
4. Run the application

## Usage
Follow the on-screen prompts to enter your numbers and the desired operator.
